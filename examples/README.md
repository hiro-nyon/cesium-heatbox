# Examples Overview / サンプル概要

CesiumJS Heatbox のサンプルは、以下のカテゴリ別ディレクトリに整理されています。

| Category / カテゴリ | Directory | 説明 |
| --- | --- | --- |
| Basic / 基本 | `basic/` | 最小構成の UI 例。初学者向け。 |
| Observability / 観測可能性 | `observability/` | パフォーマンスオーバーレイや適応制御のメトリクス確認。 |
| Rendering / 描画 | `rendering/` | ワイヤーフレーム・高さ表現・v0.1.12 以降の描画機能。 |
| Outlines / 枠線 | `outlines/` | 標準枠線・インセット・エミュレーション比較。 |
| Selection & Limits / 選択戦略 | `selection-limits/` | density/coverage/hybrid 戦略や描画上限の調整。 |
| Data / データ処理 | `data/` | エンティティフィルタリング等の前処理ユーティリティ。 |

各カテゴリの README に、目的・主要オプション・注意点を日英併記で記載しています。

> Cesium 初期化の標準設定（Ion OFF + OSM/Carto + EllipsoidTerrain）は ADR-0012 を参照してください。
